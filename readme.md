# hikvision-ddns

基于Node.js+海康IPServer协议+Dnspod API实现的海康设备动态域名解析

## Installation

1. install node.js
2. Clone this repository:
  `git clone https://github.com/atuxe/hikvision-ddns`
3. Install the project using `npm`:
  `npm install`

## Usage

Here's a short explanation how to use `hikvision-ddns`:

* node index.js
